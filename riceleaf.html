Rice Leaf Disease Detection using Machine Learning Techniques

Contributors

Kawcher Ahmed, Tasmia Rahman Shahidi, Syed Irfan, Sifat Momen

Category

Machine Learning Model

Source Document

To ensure healthy and proper growth of the rice plants it is essential to detect any disease in time and prior to applying required treatment to the affected plants. Since manual detection of diseases costs a large amount of time and labour, it is inevitably prudent to have an automated system. This project presents a rice leaf disease detection system using machine learning approaches.
