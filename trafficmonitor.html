Traffic Monitor App Android Application

CSE 299 Junior Project Design
Associated with North South University

Contributors
Syed Md. Irfanul Alam, Md. Kamrul Hassan Kafi, Md. Touhidur Rahman, Shadman Sadique Swachchha, Rajesh Palit

Description
  • In the road of Dhaka city there is no way to detect if someone breaking the traffic rule of speed limit.
  • The main purpose of this app is to detect and notify traffic police if someone is breaking the rule.
  • By using GPS and creating speed meter an android app has been made at Android Studio in Java programming language. Google Firebase has used to store data of drivers and their details acitivity.
  • The app shows the current speed of vehicle and notify instantly when it exceed the threshold speed.

Technologies
Android Studio, Java, GPS, Speed Meter, Firebase
